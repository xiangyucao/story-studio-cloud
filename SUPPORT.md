# Support and community

Story Studio Cloud is a free, community-supported open-source project. Please use the channel that best matches your message.

## Questions and general conversation

Use [GitHub Discussions](https://github.com/xiangyucao/story-studio-cloud/discussions) for:

- Questions about using Story Studio Cloud
- Writing workflows and model-integration ideas
- Early or open-ended feature ideas
- Sharing what you created
- General project conversation

## Bugs and specific feature requests

Use [GitHub Issues](https://github.com/xiangyucao/story-studio-cloud/issues) for:

- Reproducible bugs
- Clearly scoped feature requests
- Documentation corrections

Choose the appropriate form and include enough detail to reproduce or understand the request. Search first to avoid duplicates.

## Privacy

Everything posted in public Issues and Discussions can be read by anyone. Never include private manuscript text, passwords, API keys, MCP access tokens, unpublished personal information, or other secrets.

## Response expectations

This project is maintained in the author's spare time. Reports and questions are welcome, but a response time or implementation date cannot be guaranteed.
